How to run:
    Set current directory to folder (cd [path to folder])
    Create virtual environment
        windows: 
            python -m -venv venv
            venv\Scripts\activate
        mac:
            python3 -m venv venv
            source venv/bin/activate (might need to be {activate.bat})
    Run app:
        python(3) app.py
